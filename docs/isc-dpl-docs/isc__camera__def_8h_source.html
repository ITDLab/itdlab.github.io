273// RAW data file save format
274//
275// |-----------------------------------|
276// | FILE HEADER(128byte) |
277// |-----------------------------------|
278// | DATA HEADER(64byte) |
279// |-----------------------------------|
280// | DATA |
281// |-----------------------------------|
282// | DATA HEADER(64byte) |
283// |-----------------------------------|
284// | DATA |
285// |-----------------------------------|
286//
287
288// FILE HEADER
289/*
290 型 サイズ(byte) データ 内容
291 0 CHAR 32 MARK "ISC VM/XC/... RAW DATA"
292 1 INT 4 VERSION ヘッダーのバージョン
293 2 INT 4 HEADER SIZE ヘッダーのサイズ（Byte)
294 4 INT 4 CAMERA MODEL Camera 0:VM 1:XC 2:4K 3:4KA 4:4KJ
295 5 INT 4 WIDTH 最大画像幅
296 6 INT 4 HEIGHT 最大画像高さ
297 3 FLOAT 4 D_INF D_INF
298 7 FLOAT 4 BF BF
299 8 FLOAT 4 Dz Dz
300 9 FLOAT 4 BASE LENGTH BASE Length
301 10 INT 4 GRAB MODE
302 0x01 Pallax
303 0x02 Correct Image
304 0x03 Befor Correct Image
305 0x04 Bayer Image
306 11 INT 4 SHUTTER MODE
307 0x00 Normal
308 0x01 Single
309 0x02 Double
310 12 INT 4 COLOR MODE Color有効 0:off 1:on
311 13 INT 4*12 RESERVE(0)
312
313 128
314*/

338// DATA HEADER
339/*
340 型 サイズ(byte) データ 内容
341 0 INT 4 VERSION ヘッダーのバージョン
342 1 INT 4 HEADER SIZE ヘッダーのサイズ（Byte)
343 2 INT 4 DATA SIZE データサイズ
344 3 INT 4 DATA COMPRESSED データは圧縮されている 0:none 1:compressed
345 4 INT 4 FRAME_INDEX Frame番号
346 5 INT 4 TYPE
347 0x01 Mono
348 0x02 Color
349 6 INT 4 STATUS AutoCalib　Staus
350 7 INT 4 ERROR CODE エラーコード
351 8 INT 4 EXPOSURE Exposure値
352 9 INT 4 GAIN Gain値
353 10 INT 4*6 RESERVE(0)
354
355 64
356*/
