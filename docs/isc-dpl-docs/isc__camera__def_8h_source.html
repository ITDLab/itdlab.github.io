274// RAW data file save format
275//
276// |-----------------------------------|
277// | FILE HEADER(128byte) |
278// |-----------------------------------|
279// | DATA HEADER(64byte) |
280// |-----------------------------------|
281// | DATA |
282// |-----------------------------------|
283// | DATA HEADER(64byte) |
284// |-----------------------------------|
285// | DATA |
286// |-----------------------------------|
287//
288
289// FILE HEADER
290/*
291 型 サイズ(byte) データ 内容
292 0 CHAR 32 MARK "ISC VM/XC/... RAW DATA"
293 1 INT 4 VERSION ヘッダーのバージョン
294 2 INT 4 HEADER SIZE ヘッダーのサイズ（Byte)
295 4 INT 4 CAMERA MODEL Camera 0:VM 1:XC 2:4K 3:4KA 4:4KJ
296 5 INT 4 WIDTH 最大画像幅
297 6 INT 4 HEIGHT 最大画像高さ
298 3 FLOAT 4 D_INF D_INF
299 7 FLOAT 4 BF BF
300 8 FLOAT 4 Dz Dz
301 9 FLOAT 4 BASE LENGTH BASE Length
302 10 INT 4 GRAB MODE
303 0x01 Pallax
304 0x02 Correct Image
305 0x03 Befor Correct Image
306 0x04 Bayer Image
307 11 INT 4 SHUTTER MODE
308 0x00 Normal
309 0x01 Single
310 0x02 Double
311 12 INT 4 COLOR MODE Color有効 0:off 1:on
312 13 INT 4*12 RESERVE(0)
313
314 128
315*/
316

339// DATA HEADER
340/*
341 型 サイズ(byte) データ 内容
342 0 INT 4 VERSION ヘッダーのバージョン
343 1 INT 4 HEADER SIZE ヘッダーのサイズ（Byte)
344 2 INT 4 DATA SIZE データサイズ
345 3 INT 4 DATA COMPRESSED データは圧縮されている 0:none 1:compressed
346 4 INT 4 FRAME_INDEX Frame番号
347 5 INT 4 TYPE
348 0x01 Mono
349 0x02 Color
350 6 INT 4 STATUS AutoCalib　Staus
351 7 INT 4 ERROR CODE エラーコード
352 8 INT 4 EXPOSURE Exposure値
353 9 INT 4 GAIN Gain値
354 10 INT 4*6 RESERVE(0)
355
356 64
357*/
